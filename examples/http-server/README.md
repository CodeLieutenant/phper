# http-server

Http server example.

Power by [tokio](https://crates.io/crates/tokio) and [hyper](https://crates.io/crates/hyper).

## Environment

```bash
# Optional, specify if php isn't installed globally.
export PHP_CONFIG = <Your path of php-config>
```

## Build

```bash
cargo build --release
```

## Test

```bash
cargo test --release
```

## Install

```bash
cargo run --release -- install
```
